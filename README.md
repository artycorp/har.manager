# HAR.MANAGER

Анализатор HAR файлов. Работает в браузере, данные не уходят на сервер.

## Возможности

- **Работает в браузере:** Все данные остаются у вас, анализ происходит локально.
- **Темный интерфейс:** Темная тема с хорошей читаемостью.
- **Waterfall диаграмма:** Разбивка времени запросов (на данный момент без DNS, Connect, SSL, Wait, Receive, только общее время запроса и его начало).
- **Интеграция с мониторингом:** Ссылки на Grafana, Loki и Coroot из заголовков запросов.
- **Фильтры:** По методам, статус кодам и путям.

## Быстрый старт (Docker)

```bash
docker-compose up -d
```
Откройте [http://localhost:8085](http://localhost:8085) в браузере.

## Разработка

```bash
npm install
npm run dev
```

## Как использовать

1. Откройте DevTools в браузере (F12).
2. Перейдите на вкладку **Network**.
3. Выполните действия, которые хотите записать.
4. Нажмите правой кнопкой мыши в списке запросов и выберите **"Save all as HAR with content"**.
5. Перетащите файл `.har` в приложение.
